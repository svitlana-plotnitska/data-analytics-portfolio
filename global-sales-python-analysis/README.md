# Global Sales Data Analysis (Python)

Python data analysis project focused on cleaning, analyzing, and visualizing global sales data.

The dataset contains information about company sales across multiple countries and sales channels.

## Dataset

The project uses three datasets:

events.csv — sales data over several years  
products.csv — product categories and codes  
countries.csv — country and region information  

## Project tasks

• Data overview and understanding of datasets  
• Data cleaning and preprocessing  
• Handling missing values and duplicates  
• Merging multiple datasets into a single dataframe  
• Analysis of key business metrics  
• Visualization of sales performance  

## Tools used

Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Google Colab
